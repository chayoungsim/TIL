# 4. Truthy & Falsy

부울값 - 진실/거짓

## always falsy:
- false
- 0 (zero)
- '' or "" (empty string)
- null
- undefined
- NaN

## Truthy
- '0' (a string containing a single zero)
- 'false' (a string containing the text “false”)
- [] (an empty array)
- {} (an empty object)
- function(){} (an “empty” function)
