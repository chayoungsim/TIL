# Promise
- 프로미스는 자바스크립트 비동기 처리에 사용되는 객체입니다.
- 프로미스는 주로 서버에서 받아온 데이터를 화면에 표시할 때 사용합니다. 일반적으로 웹 애플리케이션을 구현할 때 서버에서 데이터를 요청하고 받아오기 위해 아래와 같은 API를 사용합니다.
```javascript
$.get('url 주소/products/1', function (response) {
  // ...
});

```
